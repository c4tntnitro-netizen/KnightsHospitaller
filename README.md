# Knights Hospitallar

A Starsector mod by caccus for Starsector 0.98a-RC8.

## Download and install

Download `KnightsHospitallar.zip` from the release page assets.

Do not use GitHub's automatic `Source code (zip)` or `Source code (tar.gz)` downloads for installation. Those archives add wrapper folders such as `KnightsHospitaller-main` or tag-named folders, which can leave the mod nested one folder too deep.

To install:

1. Download `KnightsHospitallar.zip` from the release assets.
2. Extract it into your Starsector `mods` folder.
3. Confirm the final path looks like this:

   `Starsector/mods/Knights Hospitallar/mod_info.json`

4. Start Starsector and enable `Knights Hospitallar` in the launcher.

If you see something like `Starsector/mods/KnightsHospitaller-main/Knights Hospitallar/mod_info.json`, you downloaded GitHub's source archive instead of the release zip.

## Mod contents

Knights Hospitallar adds a Luddic humanitarian order centered on the Abundant Mercy, a unique Hospitaller Invictus stationed at Gilead.

Current contents include:

- A Kanta's Den bar encounter introducing the Knights Hospitaller.
- The Abundant Mercy fleet orbiting Gilead.
- Knight-Surgeon Gabriel Malachi as the fleet commander.
- A dialogue tree tied to Luddic quest progress and the Planetkiller choice.
- The Great Labor quest hook and reward flow.
- The unique `Abundant Mercy` Invictus.
- The `Gethsemane Rescue Bay` built-in hullmod, reducing fleetwide crew and fighter casualties by 80%.
- The Rapid Dragon, a militarized Buffalo-derived support ship.
- The Corinthian fighter wing.
